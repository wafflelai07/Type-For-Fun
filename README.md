# Type For Fun
It is a simple typing practice website built for fun.

Test your typing speed, accuracy, and time performance in a clean and minimal interface.

# Purpose
This project is designed for:
- Practicing your typing skills
- Understand your typing speed
- Tracking accuracy
Having a quick and fun typing experience
### Most importantly: HAVE FUN IN TYPING!!!
No login, no data storage — just pure typing practice.
